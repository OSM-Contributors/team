# Open Stand Model
Open Stand Model (OSM) is a tree list modelling platform (Application Programming Interface [API]) for developing regionally calibrated individual-tree models that project stand dynamics. This is a collaborative open-source project involving an increasing number of public and private organizations and individual researchers in North America. 

# OSM - Teams Repository
A public-facing repository of OSM Projects, Issues, Discussions, Releases, and Documentation. No code lives here.
